# FullData &mdash; Pricing Mock

Standalone Vite + React 19 + TypeScript + Tailwind 4 + Framer Motion demo that renders the FullData pricing model (7 bandas + Custom + roadmap add-ons) per `clients/fulldata/out/04_executive_document.md` &sect;3.1.

## Dev

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Build

```bash
npm run build
```

Outputs to `dist/`. Serve with `npm run preview` for a production smoke check.
